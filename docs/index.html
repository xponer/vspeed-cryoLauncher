Everything in one place

Built for modpack players

No fiddling with Java versions, memory flags, or loader installs. Cryo handles it.

📦

Install packs in a click

Browse Modrinth (and CurseForge) right inside the launcher and install any modpack as a new instance.

⚙️

Every loader, no setup

NeoForge, Forge, Fabric, Quilt or vanilla — the engine downloads and installs the right one for you.

☕

Java, auto-detected

Cryo finds the Java a pack needs and uses it automatically — or pick from every JRE on your PC.

⚡

One-click Optimize

Sizes your RAM and GC flags from the pack's mods and your actual hardware. No more guessing.

🔄

Always up to date

Tiny delta auto-updates ship new features and fixes in the background — nothing to reinstall.

🧊

Faster launches

The VSpeed engine caches startup work (AppCDS) so your packs boot quicker the next time around.

Your account, your machine

Login you can actually trust

After the launcher-stealer scares, security isn't optional. Cryo is open-source, and your sign-in never leaves your PC in the clear.

Encrypted at rest. Tokens are sealed with Windows DPAPI — only your Windows user on your PC can read them.
Never sees your password. You sign in through Microsoft's own window — Cryo only gets a token.
Open-source. Every line is on GitHub — read it, build it, or audit it yourself.
